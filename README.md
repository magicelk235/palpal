# PalPal

PalPal converts image files into assembly `db` format byte arrays.

## Requirements

- Python
- pip packages: `Pillow` and `tkinter`

## Install dependencies

```bash
pip3/pip install Pillow
```

## Run
```bash
python3/python palpal.py
```
## Usage

1. Click Open File and select any image
2. Click Convert to convert it to assembly `db` byte rows
3. Click Copy to copy the output
